
# nBank

Aplikacja dla klientów fikcyjnego banku nBank.



## Technologie:

- **Flutter**
- **Firebase**


## 10 kwietnia

- Podstawowe UI

- Lista transkacji

- Ekran logowania

## 8 maj

- Współpraca z chmurą lub lokalną bazą danych (do ustalenia)

## 5 czerwiec

- Przelewy pomiędzy użytkownikami w banku

- Analiza historii wydatków

- Budżet
